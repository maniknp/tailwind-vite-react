
- https://heroicons.com/

**Installation**
```bash
npm install @heroicons/react
```
Now each icon can be imported individually as a React component:

```react
import { BeakerIcon } from '@heroicons/react/24/solid'

function MyComponent() {
  return (
    <div>
      <BeakerIcon className="size-6 text-blue-500" />
      <p>...</p>
    </div>
  )
}
```
